Accessibility-Standars

The Web Accessibility Initiative, project by the World Wide Web Consortium, published the Web Content Accessibility Guidelines. Then, the WAI released the WCAG 2.0. WCAG 2.0 aims to be up to date and more technology neutral. Though web designers can choose either standard to follow, the WCAG 2.0 have been widely accepted as the definitive guidelines on how to create accessible websites.
